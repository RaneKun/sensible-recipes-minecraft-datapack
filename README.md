# Sensible Recipes — Collection

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)
[![Minecraft](https://img.shields.io/badge/Minecraft-1.20.1-brightgreen.svg)](https://www.minecraft.net/)
[![Pack Format](https://img.shields.io/badge/Pack%20Format-15-blue.svg)](https://minecraft.wiki/w/Data_pack)

A collection of Minecraft datapacks and a companion resource pack that overhaul vanilla (and modded) crafting recipes to be more intuitive, less wasteful, and more rewarding — centered around the concept of the **Unicutter**, an expanded use of the stonecutter block.

---

## 📦 Contents

| Project | Type | Description |
|---|---|---|
| [`sensible_recipes`](./sensible_recipes/) | Datapack | Vanilla-only recipe improvements |
| [`sensible_recipes_plus`](./sensible_recipes_plus/) | Datapack | Vanilla + mod recipe improvements |
| [`stonecutter_to_unicutter`](./stonecutter_to_unicutter/) | Resource Pack | Renames Stonecutter → Unicutter in-game |

---

## 🧩 Which Pack Should I Use?

- **Playing vanilla or a lightly modded world?** → Use [`Sensible Recipes`](./sensible_recipes/)
- **Playing a modpack with supported mods?** → Use [`Sensible Recipes +`](./sensible_recipes_plus/) *(replaces the base pack — don't use both)*
- **Want the Unicutter name in-game?** → Always pair either datapack with [`Stonecutter to Unicutter`](./stonecutter_to_unicutter/)
